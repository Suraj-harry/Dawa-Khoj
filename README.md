# 💊 Dawa-Khoj

Dawa-Khoj is a medicine search and recommendation system that helps users find similar medicines based on their search query. The project combines a React frontend with a Python backend and a machine learning model to provide medicine recommendations quickly and accurately.

The main objective of this project is to make medicine searching easier by suggesting alternative medicines with similar properties, helping users and pharmacists compare available options.

---

## Features

### User Features

- Search medicines by name

- Machine Learning based recommendation

## Installation

### Clone Repository

```bash
git clone https://github.com/Suraj-harry/Dawa-Khoj.git
```

Move into the project folder

```bash
cd Dawa-Khoj
```

---

### Backend Setup

Install Python dependencies

```bash
pip install -r requirements.txt
```

Run the Flask server

```bash
python app.py
```

---

### Frontend Setup

Move to frontend directory

```bash
cd frontend
```

Install dependencies

```bash
npm install
```

Start the React application

```bash
npm start
```

---

## Working

1. User enters the medicine name.
2. The request is sent to the Flask backend.
3. The machine learning model processes the query.
4. Similar medicines are identified.
5. The backend returns the results.
6. React displays the recommendations on the screen.
